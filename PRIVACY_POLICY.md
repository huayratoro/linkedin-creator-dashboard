# Política de Privacidad para "Personal Engagement API"

**Fecha de la última actualización:** 25 de Noviembre de 2025

## 1. Introducción y Alcance
Esta Política de Privacidad describe cómo se gestiona la información dentro de la aplicación **"Personal Engagement API"**.

Esta herramienta es un proyecto **no comercial** desarrollado por Marcos Benedicto y está destinada **única y exclusivamente para uso personal e interno** del desarrollador (el titular de la cuenta de LinkedIn) con el objetivo de crear un *dashboard* privado para monitorear métricas propias de creador de contenido en LinkedIn.

## 2. Recolección y Uso de Datos
Dado su propósito de uso interno, la aplicación sigue una estricta política de **No Recolección de Datos de Terceros**:

* **Datos No Recolectados de Terceros:** La aplicación **no está diseñada para interactuar con otros usuarios** de LinkedIn ni para recopilar, almacenar o procesar información personal (como nombres, direcciones de correo electrónico, etc.) de miembros de LinkedIn distintos al desarrollador.
* **Datos Procesados (Datos del Desarrollador):** La aplicación accede a la información del desarrollador (titular de la cuenta) a través de la API oficial de LinkedIn únicamente para obtener y monitorear:
    * Métricas propias de contenido (vistas, reacciones, comentarios, etc.).
    * Información asociada al perfil del desarrollador necesaria para la API.

Estos datos se utilizan **exclusivamente** para alimentar un dashboard de uso personal y para proveer la funcionalidad solicitada por el desarrollador.

## 3. Almacenamiento y Seguridad de los Datos
* **Uso Interno:** Todos los datos obtenidos son para el consumo interno y **no se almacenan en servidores de terceros** con fines de distribución pública. La aplicación puede almacenar los datos en la infraestructura personal del desarrollador.
* **Seguridad:** La seguridad de los datos depende de las prácticas y el entorno local donde el desarrollador aloje la aplicación y el dashboard.

## 4. Distribución y Compartición de Datos
La aplicación "Personal Engagement API" **no distribuye, vende ni comparte** ningún dato personal o métrica con terceros. Los datos son para el consumo privado y exclusivo del desarrollador.

## 5. Servicios de Terceros
La aplicación depende de la **API oficial de LinkedIn** para acceder a los datos. Al utilizar la aplicación, usted también está sujeto a los términos de servicio y la política de privacidad de LinkedIn.
